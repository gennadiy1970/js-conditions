Сравнение: == vs ===

1 == true  // true
1 === true  // false
0 == false  // true
0 === false  // false
5 == 5  // true
5 === 5  // true

typeof

typeof 1  // number
typeof 0  // number
typeof true  // boolean
typeof false  // boolean

=== это две проверки:

  1. проверка на значения == // true or false
  2. проверка на тип // number or boolean
