Тернарый опрератор

Если вам нужно вернуть значение, то требуется ОПЕРАТОР а не ВЫРАЖЕНИЕ

Тернарный оператор operand1 ? operand2 : operand3

let answer = boolean ? true : false ;

Переменной answer будет присвоено значение operand2 (т.е. после ?) если operand1 вернет true

Переменной answer будет присвоено значение operand3 (т.е. после :) если operand1 вернет false


let num = 10;
let result = num > 0 
    ? 'да, больше'
    :'нет, меньше';
'да, больше' 
    

let num = Math.random();
let result = num > 0.5 
    ? 1
    : 0;
 result = 1 или 0
